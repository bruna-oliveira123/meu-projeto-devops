# Projeto de Fluxo CI/CD - Disciplina de DevOps

Este repositório foi criado como parte das atividades práticas da disciplina de DevOps. 
O objetivo é demonstrar a configuração de um fluxo de integração e entrega contínua utilizando o GitHub.

## O que este projeto contém:
* Práticas de criação de branches e commits.
* Gerenciamento de Pull Requests (PRs).
* Organização de arquivos em Markdown.
